<h1><b>🕷️ CrawView</b></h1>

<b>OFFLINE SCAM DETECTION AND MESSAGE ANALYSIS TOOL</b><br>
<b>ANALYZES SMS, EMAILS, AND CHAT MESSAGES</b><br>
<b>USING RULE-BASED DETECTION, PATTERN MATCHING,</b><br>
<b>AND CONTEXT-AWARE RISK ANALYSIS.</b>

</div>

---

<div align="center">

## <b>❓ OVERVIEW</b>

</div>

<b>ONLINE SCAMS CONTINUE TO TARGET USERS THROUGH SMS, EMAILS,</b>  
<b>AND MESSAGING PLATFORMS USING SOCIAL ENGINEERING TECHNIQUES.</b>

<b>CRAWVIEW HELPS IDENTIFY SUSPICIOUS CONTENT BY ANALYZING</b>  
<b>MESSAGES LOCALLY WITHOUT SENDING DATA TO EXTERNAL SERVICES.</b>

<b>SUPPORTED DETECTION INCLUDES:</b>

<b>- PHISHING MESSAGES</b>  
<b>- FAKE KYC REQUESTS</b>  
<b>- DIGITAL ARREST SCAMS</b>  
<b>- FAKE REFUND MESSAGES</b>  
<b>- ROMANCE SCAMS</b>  
<b>- INVESTMENT AND CRYPTO SCAMS</b>

---

<div align="center">

## <b>🚀 KEY FEATURES</b>

</div>

<b>✔ COMPLETELY OFFLINE ANALYSIS</b>  
<b>✔ NO CLOUD OR EXTERNAL API REQUIRED</b>  
<b>✔ RISK SCORE GENERATION</b>  
<b>✔ URL, EMAIL, PHONE, UPI, AND WALLET EXTRACTION</b>  
<b>✔ LANGUAGE SUPPORT (ENGLISH, HINDI, HINGLISH)</b>

<b>✔ LOCAL REPUTATION MEMORY</b>  
<b>✔ PRIVACY-FOCUSED DESIGN</b>

---

<div align="center">

## <b>🧠 HOW IT WORKS</b>

</div>

<b>1. USER PASTES A MESSAGE</b>

<b>2. CRAWVIEW EXTRACTS IMPORTANT ENTITIES</b>

<b>3. DETECTION RULES ANALYZE THE CONTENT</b>

<b>4. SUSPICIOUS PATTERNS ARE IDENTIFIED</b>

<b>5. A RISK SCORE IS CALCULATED</b>

<b>6. A FINAL VERDICT IS DISPLAYED</b>
